This is an Android Mobile Tower Defense game made with Unity3D. I fully released it around September of 2019 and it has a good years worth of work thrown into it. A co-developer joined me halfway through that has an AA in Software Engineering. I'd say that 97% of all assets and 70% of scripting was done by me. Modeling was done with MagicaVoxel and then converted properly in Blender for use in Unity.

{% include image.html url="https://play.google.com/store/apps/details?id=com.JazzApps.VoxelVanquish&hl=en_US" image="projects/voxelvanquish/voxelvanquish.jpg" %}

There is a PC version in the works that is already leaps ahead of the mobile version that I released in 2019. However, I am not showcasing this PC version until release (possibly in mid 2021). The reason for delays is the difficulty of converting all of the singleplayer code into multiplayer netcode. We plan on using Photon Engine for easy scalability, but it's quite the endeavor for a team of 2.
